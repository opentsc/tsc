# 白皮书更新日志 / Whitepaper Changelog

TSC（薄壳公司）白皮书是"魂"——一部组织之法。它的版本演进与"壳"
（[opentsc/opentsc](https://github.com/opentsc/opentsc) 的实现）独立推进。

---

## [3.5] — 当前 ·《魂》

把 v3.0 劈成更清楚的两半：**魂（法 + 判断 + 记忆，可移植）/ 壳（运行容器，可替换）**。

- 确立"魂壳分离"为核心命题：魂能从一个壳完整倒出、注进另一个壳复活（组织层面的忒修斯之船）。
- 十二法则成形：存在（创世不可变 / 魂壳分离 / 按 ID 立身）、感知（事件只增 / 先证据后判断 / 先草稿后入册）、判断（双法典 / 现实校准 / 属性来自事件）、进化（按需生长 / 四重自创生 / 递归同构）。
- 判断法典（什么算好 / 可信 / 值得动）显式化、可审计。
- 文件：[`whitepaper/TSC-3.5-白皮书-魂.md`](whitepaper/TSC-3.5-白皮书-魂.md)。
