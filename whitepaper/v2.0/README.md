## 本版本核心创新

v2.0 在 v1.0 的基础上，引入了一个根本性的突破：

**AI Agent 从"工具"升格为 TSC 的"第五类正式成员"。**

这不是文字游戏，而是组织设计的范式转变——当 AI 是成员而不是工具，整个治理逻辑、贡献归因、角色设计都要重新定义。
